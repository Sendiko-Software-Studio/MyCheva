# Announcement

Package announcement ini diisi oleh file file yang berkaitan dengan fitur Post Announcement.

## Struktur Package: 

```
\announcement
    \data
    \domain
    \presentation
```
<br>
Package data digunakan untuk menyimpan file yang berkaitan dengan data/model, seperti data class.
<br><br>
Package domain digunakan untuk menyimpan file yang tidak berkaitan dengan data maupun user interface, seperti menjadi penengah diatara package data dan presentation.
<br><br>
Package presentation digunakan untuk menyimpan file yang berkaitan dengan user interface, seperti komponen Composables, Screen, State, dan ViewModel.
